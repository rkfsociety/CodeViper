# Правила проекта CodeViper

## Дорожная карта (ROADMAP.md)

- Раздел «📋 В планах» — задачи **сгруппированы по теме**, внутри каждой группы отсортированы **от лёгких к сложным**.
- **Сквозная нумерация всех пунктов планов начинается строго с 1** и идёт без пропусков до конца раздела (группы не сбрасывают счётчик — нумерация продолжается между группами).
- Когда пункт выполнен и переезжает в «✅ Сделано» — **обязательно перенумеровать оставшиеся пункты планов заново с 1**, чтобы не возникало дыр (например, после удаления пунктов 1–3 список не должен начинаться с 4).
- При переносе/удалении пунктов проверять и обновлять внутренние ссылки между ними (например, «из п. N»).
- В разделе «✅ Сделано» номера не используются.
