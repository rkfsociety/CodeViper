### 🧭 Встроенный оркестратор (embedded model)

Лёгкая модель (~1 ГБ), вшитая в агента — работает без Ollama и LM Studio. Разбирает намерение пользователя, строит план и перефразирует запрос для основной модели. Суть запроса не меняет.

- [ ] **40. Движок `node-llama-cpp` в Electron** — установить `node-llama-cpp`; скомпилировать нативный модуль под win32/linux/darwin в `postinstall`; smoke-тест: загрузить GGUF, получить токен, выгрузить
- [ ] **41. Загрузчик модели оркестратора** — при первом старте скачивать `Qwen2.5-1.5B-Instruct.Q4_K_M.gguf` (~900 МБ) в `userData/orchestrator/`; прогресс-бар в UI; кнопка «Скачать оркестратор» в настройках; хранить путь в `config.json`
- [ ] **42. Класс `OrchestratorModel`** — singleton в main process; методы `analyze(userMessage): { plan: string[], rephrased: string, isComplex: boolean }`; системный промпт: «Ты планировщик. Составь шаги выполнения задачи. Перефразируй запрос для исполнителя — кратко, конкретно, без потери смысла»; использует `node-llama-cpp` без стриминга (только итог)
- [ ] **43. Интеграция в `AgentRunner`** — перед `prepareAgentRunContext` вызывать `orchestrator.analyze(userMessage)`; результат добавлять в системный промпт блоком `## План оркестратора`; если `isComplex=true` — использовать перефразированный запрос вместо исходного как подсказку для основной модели (исходный сохраняется в истории как есть)
- [ ] **44. UI: индикатор «Планирую...»** — stream-событие `orchestrating` в `AgentStreamPayload`; в чате показывать серый чип «🧭 Составляю план...» пока оркестратор думает; после — скрывать или показывать раскрывающийся блок с планом
- [ ] **45. Настройки оркестратора** — тумблер «Включить оркестратор» в настройках (вкладка Модель); порог `minMessageLength` (по умолчанию 80 символов) — короткие сообщения пропускаются без анализа; кнопка «Удалить модель» для освобождения места
- [ ] **46. Поддержка альтернативных моделей** — в настройках кнопка «Выбрать свою модель» → `dialog.showOpenDialog` с фильтром `.gguf`; путь сохраняется в `config.json`; при старте проверяется существование файла
