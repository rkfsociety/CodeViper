Вы описываете задачу в чате — агент сам читает файлы, правит их и запускает команды в терминале.

## Быстрый старт

```powershell
cd app && npm install   # первый раз
```

Запуск: двойной клик на **`CodeViper.cmd`**. С консолью: `CodeViper.cmd console`

**Требования:** Windows 10/11, Node.js 18+, 8 ГБ RAM.  
Для локальных моделей — [Ollama](https://ollama.com). Для облака — API key DeepSeek или OpenAI.
