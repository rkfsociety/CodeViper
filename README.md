# CodeViper

**CodeViper** — локальный AI-агент с GUI для программирования. Работает полностью на вашем компьютере через [Ollama](https://ollama.com). Никаких подписок, API-ключей и облачных лимитов.

Вы описываете задачу в чате — агент сам читает файлы, правит их и запускает команды в терминале.

![stack](https://img.shields.io/badge/Electron-36-47848F) ![stack](https://img.shields.io/badge/React-19-61DAFB) ![stack](https://img.shields.io/badge/Ollama-local-7EE787)

## Требования

| | |
|---|---|
| Windows 10/11 | ✅ |
| RAM | 8 GB (16 GB лучше) |
| Node.js | 18+ |
| [Ollama](https://ollama.com) | запущен в трее |
| Модель | `qwen2.5-coder:7b` или новее |

## Быстрый старт

**Первый раз:**
```powershell
cd app && npm install
```

**Запуск:** двойной клик по **`CodeViper.cmd`** в корне репозитория.  
С консолью: `CodeViper.cmd console`

После запуска:
1. **«+ Чат»** — создать чат, выбрать папку проекта
2. Убедиться что индикатор Ollama **зелёный**
3. Скачать модель (если нет) в **Настройках**
4. Описать задачу

## Возможности

- **Чат-агент** — читает файлы, правит, запускает команды; ответы потоком с Markdown
- **Параллельные tool calls** — независимые операции чтения выполняются одновременно
- **Автовыбор модели** — подбирает нужную под сложность задачи; минимум 7B при сложности ≥60
- **Эскалация модели** — при рефьюзале автоматически переключается на более тяжёлую
- **Семантический поиск памяти** — векторный поиск через `nomic-embed-text`
- **Папки чатов с проектом** — папка хранит привязанный projectPath, новые чаты наследуют его
- **Самообучение** — запоминает уроки и навыки между сессиями
- **Саморедактирование** — агент может улучшать собственный код
- **История чатов** — папки, поиск, drag-and-drop
- **Режимы доступа** — от «спрашивать всё» до «без подтверждений»
- **Суммаризация контекста** — при ≥85% лимита сжимает историю через Ollama
- **Структурированный лог** — NDJSON по каждому шагу в `%APPDATA%/CodeViper/logs/`
