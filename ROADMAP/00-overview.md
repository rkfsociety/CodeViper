# Дорожная карта CodeViper

Актуальные задачи проекта. Выполненное — [ROADMAP_DONE.md](ROADMAP_DONE.md). Назад в [README](README.md).

### Формат задач

**Шаблон пункта:**

```text
N · [S/M/L/XL] · Краткое название — уровень 1|2|3|4
- Цель: один измеримый результат
- Файлы: конкретные пути (app/electron/main/…, app/src/…)
- Действие: одна атомарная правка
- Проверка: npm run typecheck | npm test -- … | сценарий в UI
```

**Правила:** пункты **1…499** выбираются **сначала по важности/нужности**, а **внутри важности — от лёгких к более сложным**.
