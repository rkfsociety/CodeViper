# Политика безопасности

## Поддерживаемые версии

Исправления безопасности выпускаются только для последней стабильной версии.

| Версия | Поддержка |
| ------ | --------- |
| 0.2.x  | ✅ Да      |
| < 0.2  | ❌ Нет     |

## Сообщить об уязвимости

**Не создавайте публичный Issue** — это раскроет уязвимость до выхода исправления.

Отправьте отчёт через [GitHub Security Advisories](https://github.com/rkfsociety/CodeViper/security/advisories/new) или на почту: **romankuzminvital@gmail.com**

Укажите в отчёте:
- Описание уязвимости
- Шаги воспроизведения
- Возможные последствия
- Версию CodeViper

**Ответ в течение 72 часов.** После подтверждения уязвимость будет исправлена и опубликован CVE (при необходимости).
